# SDS Date Plugin / Plugin de Date SDS

## English
### Description
A utility plugin to quickly update date fields within JSON files to the current timestamp.

### Features
- **Smart Detection**: Automatically offers the update option when right-clicking on a value associated with a key named "date".
- **ISO Format**: Generates timestamps in UTC ISO format (e.g., `2023-10-27T10:00:00.000Z`).

### Usage
1. Position your cursor on the value of a "date" field in the JSON editor.
2. Right-click to open the context menu.
3. Select "Update date" to replace the value with the current UTC timestamp.

## Français
### Description
Un plugin utilitaire pour mettre à jour rapidement les champs de date dans les fichiers JSON avec l'horodatage actuel.

### Fonctionnalités
- **Détection Intelligente** : Propose automatiquement l'option de mise à jour lors d'un clic droit sur une valeur associée à une clé nommée "date".
- **Format ISO** : Génère des horodatages au format ISO UTC (ex: `2023-10-27T10:00:00.000Z`).

### Utilisation
1. Placez votre curseur sur la valeur d'un champ "date" dans l'éditeur JSON.
2. Faites un clic droit pour ouvrir le menu contextuel.
3. Sélectionnez "Mettre à jour la date" pour remplacer la valeur par l'horodatage UTC actuel.
