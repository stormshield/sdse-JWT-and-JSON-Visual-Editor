# Boolean Plugin / Plugin de Booléens

## English
### Description
A simple utility to quickly toggle boolean values (`true` / `false`) in the JSON editor.

### Features
- **Quick Toggle**: Detects boolean values under the cursor and offers the opposite value in the context menu.

### Usage
1. Right-click on a `true` or `false` value in the editor.
2. Select the opposite value (e.g., if it's `true`, select `false`) to replace it instantly.

---

## Français
### Description
Un utilitaire simple pour basculer rapidement les valeurs booléennes (`true` / `false`) dans l'éditeur JSON.

### Fonctionnalités
- **Bascule Rapide** : Détecte les valeurs booléennes sous le curseur et propose la valeur opposée dans le menu contextuel.

### Utilisation
1. Faites un clic droit sur une valeur `true` ou `false` dans l'éditeur.
2. Sélectionnez la valeur opposée (ex: si c'est `true`, sélectionnez `false`) pour la remplacer instantanément.
