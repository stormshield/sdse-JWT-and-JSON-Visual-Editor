# Plugin Extractor

## Description
(EN) This plugin allows extracting specific JSON elements (subtrees) from the application and saving them to an `extract.json` file.
(FR) Ce plugin permet d'extraire des éléments JSON spécifiques (sous-arbres) depuis l'application et de les sauvegarder dans un fichier `extract.json`.

## Usage
1.  Right-click on a node in the Tree View.
2.  Select "Extract to extract.json".
3.  The selected element will be merged into the `extract.json` file in the same directory as the application.
